Calendario Editoriale

Il file deve girare su https:// — usa GitHub Pages, gratuito e pronto in 5 minuti.

⚠️ Perché non funziona dal PC?Il browser blocca chiamate internet da file locali. GitHub Pages ti dà un link https:// che funziona ovunque.
1
Crea account GitHub

Vai su github.com/signup

2
Crea repository

Vai su github.com/new → chiama il repo calendario → spunta Public → Create.

3
Carica come index.html

Clicca uploading an existing file → carica questo file → rinominalo index.html → Commit changes.

4
Attiva GitHub Pages

Settings → Pages → Branch: main → Save. Aspetta 1-2 minuti.

5
Apri il link

https://tuonome.github.io/calendario

Connetti Supabase

Inserisci le credenziali del tuo progetto.

Settings → General → Project URL
Settings → API Keys → Publishable key
