Calendario Editoriale

Per funzionare con Supabase il file deve essere su https://. Usa GitHub Pages — gratuito, 5 minuti.

⚠️ Perché non funziona aprendo dal PC?Il browser blocca chiamate a internet da file locali. Con GitHub Pages ottieni un link https:// che funziona ovunque.
1
Crea account GitHub

Vai su github.com/signup

2
Crea repository

Vai su github.com/new → chiama il repo calendario → spunta Public → Create.

3
Carica come index.html

Clicca uploading an existing file → carica questo file → rinominalo index.html → Commit changes.

4
Attiva GitHub Pages

Settings → Pages → Branch: main → Save. Aspetta 1-2 minuti.

5
Apri il link

https://tuonome.github.io/calendario

Connetti Supabase

Inserisci le credenziali del tuo progetto Supabase.

Settings → General → Project URL
Settings → API Keys → Publishable key
